SPEC CPU result comparison

28nd July 2020

554.roms_r

ROMS is a Regional Ocean Modeling System, a free-surface, hydrostatic, primitive equation model discretized with a terrain-following vertical coordinate system. Forecasts include predictions of water temperature, ocean currents, salinity, and sea surface height.

Official description is available at https://www.spec.org/cpu2017/Docs/benchmarks/554.roms_r.html.

DISCLAIMER: All the SPEC results are "non-reportable". The measurements were taken in order to compare different compilers, not different hardware, please do not attempt to make any comparisons of the different machines used. Moreover, we have broken various SPEC benchmarking rules, we have even patched some benchmark sources to be able to compile them with -fstrict-aliasing, build with -fno-common, and so forth. And the data may be just wrong, the scripts generating all the fancy graphs and tables have been known to contain bugs in the past. On the other hand, this is the data that we use to evaluate where GCC stands and we believe that overall this comparison of compilers is a useful one.
